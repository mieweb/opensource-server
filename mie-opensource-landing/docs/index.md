# Opensource at MIE

A centralized place for all things open source at Medical Informatics Engineering (MIE).

The Opensource Proxmox Cluster provides free, shared virtualization infrastructure
for building, testing, and deploying applications using LXC containers and
GitHub Actions CI/CD.
