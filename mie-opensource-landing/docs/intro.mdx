# Introduction

The **Opensource Proxmox Cluster** provides free, shared virtualization infrastructure for building, testing, and deploying applications using LXC containers and GitHub Actions CI/CD.

- **[Contribute →](/docs/developers/system-architecture)** — Architecture, automation internals, development workflows
